# Station Météo - LOG3500

Application web pour consulter la météo en temps réel.

## À propos

Ce projet a été réalisé dans le cadre du cours LOG3500 (Conception et programmation de sites Web I) à l'ISTEAH. L'application permet de rechercher une ville et d'obtenir ses conditions météorologiques actuelles.

**Option choisie :** Option 2 (Station Météo)

## Fonctionnalités

- Recherche de ville par nom
- Affichage de la température en °C
- Vitesse du vent en km/h
- Conditions météo en texte clair (Ensoleillé, Nuageux, Pluvieux...)
- Mise à jour en temps réel
- Design adapté aux mobiles et tablettes

## Technos utilisées

- HTML5 (balises sémantiques)
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (async/await, fetch)
- API Open-Meteo (Geocoding + Weather)

## Captures d'écran

Quelques aperçus de l'application :

- Page d'accueil avec le formulaire
- Résultat pour Jacmel (Haïti)
- Gestion des erreurs (champ vide, ville introuvable)
- Version mobile et tablette
